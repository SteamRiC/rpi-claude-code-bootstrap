# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Purpose

Bootstrap script for installing Claude Code with Haiku 4.5 on Raspberry Pi OS Lite (ARM64). Designed for fresh Pi installations to get Claude ready to help configure the system for any purpose.

## Code Style

- Indentation: 2 spaces (no tabs)
- No trailing whitespace
- Newline at EOF
- Bash: Use `set -e` for fail-fast behavior
